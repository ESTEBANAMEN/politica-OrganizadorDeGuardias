Política de Privacidad

Esta aplicación, Organizador De Guardias, no recopila, almacena ni comparte datos personales de los usuarios.

Información de uso

La app puede usar servicios de terceros (como Google AdMob) que pueden recopilar información de uso anónima, necesaria para mostrar anuncios.

Publicidad

Utilizamos Google AdMob para mostrar anuncios. AdMob puede usar identificadores del dispositivo para personalizar la publicidad. Para más información, consulte la Política de Privacidad de Google.

Permisos

La aplicación puede requerir permisos básicos del dispositivo (como acceso a almacenamiento interno) exclusivamente para funcionalidades locales. No se transmiten datos a servidores externos.

Contacto

Si tenés alguna duda o consulta sobre esta política, podés escribir a: organizador.guardias@gmail.com
